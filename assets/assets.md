# Спецификация ассетов игры «Дипломный клик»

Все файлы должны быть помещены в папку 'assets/images/'.
Поддерживаемые форматы: `.png ` (рекомендуется, с альфа-каналом для прозрачности), `.jpg` (для фонов без прозрачности).

---

## 1. Фоны локаций

| Имя файла               | Размер (ШxВ) | Описание |
|-------------------------|--------------|----------|
| `cabinet_bg.png`        | 1080 x 720   | Фон кабинета (преподаватель, доска, дверь). |
| `corridor_bg.png`       | 1080 x 720   | Фон коридора (соединение кабинета и тьюторской). |
| `tutor_bg.png`          | 1080 x 720   | Фон тьюторской (тьютор за столом). |

## Примечания

- Все изображения должны быть в стилистике 2D, спокойные цвета, персонажи в аниме/комикс стиле.
- Фоны могут быть нарисованы с учётом размещения персонажей (например, учитель уже нарисован на `cabinet_bg.png`). В этом случае отдельный файл `teacher.png` не обязателен.
- Для прозрачных элементов (иконки, персонажи) обязательно использовать `.png` с альфа-каналом.
- Если файл не найден, игра использует цветную заглушку (указано в коде `resources.py`).
